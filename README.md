# SaaS Boilerplate

Template aplikasi web SaaS modern — React + TypeScript + Vite + Tailwind.

## Quick Start

```bash
npm install
# buat .env.local dengan VITE_API_URL=http://localhost:3001
npm run dev
```

## Dokumentasi

```bash
cd docs
npm install
npm run dev   # buka http://localhost:5173
```

Dokumentasi lengkap tersedia di folder `docs/` — mencakup panduan auth, routing, design system, dan semua UI component.
